BioCDocumentc back to ToC or Class ToC

IRI: http://purl.org/bioc/Document

Each bioc:Document in the bioc:Collection.nnAn id, typically from the original corpus, identifies the particular document. It includes bioc:Passages in the document and possibly bioc:Relations over annotations on the document.
is in domain of
id dp, passages op
is in range of
infons op

BioCInfonc back to ToC or Class ToC

IRI: http://purl.org/bioc/Infon

A bioc:Infon elements is a simply a URI-based key/value pair linking the annotation to any semantic web annotation. This is deliberately open-ended URI to provide maximum expressibilty. This is equivalent to the original use of the 'infons' field across multiple classes within the BioC standard.
is in domain of
value dp

One passage in a bioc:Document.nn This might be the bioc:text in the passage and possibly bioc:Annotations over that text. It could be the bioc:Sentences in the passage. In either case it might include bioc:Relations over annotations on the passage.

BioCSentencec back to ToC or Class ToC

IRI: http://purl.org/bioc/Sentence

A bioc:Document offset to where the sentence begins in the bioc:Passage. This value is the sum of the passage offset and the local offset within the passage.
is in domain of
offset dp
is in range of
sentences op
